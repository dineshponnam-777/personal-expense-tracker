# 💰 Personal Expense Tracker with Data Visualization

## 📌 Overview

A Python-based application that helps users track personal expenses, analyze spending behavior, and visualize insights using an interactive dashboard.

This project simulates real-world financial tracking using CSV data and provides actionable insights through data analysis and visualization.

---

## 🚀 Features

* 📥 Import expense data (CSV / synthetic data)
* 🧹 Data cleaning and preprocessing
* 🗄️ SQLite database integration (persistent storage)
* 📊 Category-wise expense analysis
* 📈 Monthly spending trends
* 💳 Payment method analysis
* 💰 Budget tracking with overspending alerts
* 🔍 Interactive filters (month, category, payment method)
* 📊 Streamlit dashboard for visualization
* 📥 Export reports (CSV)

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy**
* **Matplotlib**
* **SQLite**
* **Streamlit**
